## Setup
Clone repo and run `./install.sh`

Install without git
```bash
cd; mkdir dotfiles; cd dotfiles; \
curl -#L https://github.com/aziz/dotfiles/tarball/master | \
tar -xzv --strip-components 1; \
./install.sh
```

## Provisioning
#### MacOS
`mac-install`

#### Ubuntu
`~/.templates/provisioning/ubuntu/ubuntu.sh`
